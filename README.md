pjp.funs
========

The goal of pjp.funs is to make available (to me) a bunch of functions

Installation
------------

You can install the `pjp.funs` package from Github with:

``` r
devtools::install_github("perezp44/pjp.funs", force = TRUE)
library(pjp.funs)
```
